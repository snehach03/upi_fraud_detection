Pipeline

How a transaction becomes a ring alert

1. Ingest UPI txn log → directed multigraph (NetworkX)
2. Compute degree, PageRank, betweenness, clustering
3. Detect short directed cycles (money loops)
4. Louvain community detection → density per cluster
5. Temporal burstiness + same-day pass-through
6. RF + GBM ensemble → fraud probability per account
7. Connected components of flagged nodes → ring report
